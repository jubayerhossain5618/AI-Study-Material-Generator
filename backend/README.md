# AI Study Material Generator Backend

A robust and scalable backend system for the **AI Study Material Generator** project, built with **Node.js**, **Express.js**, and **MongoDB**.

The backend provides RESTful APIs for:

* 🔐 User Authentication and Authorization (JWT)
* 📄 Document Upload and Management
* 🤖 AI-Powered Study Material Generation
* 💬 AI Chatbot Interaction
* 👨‍💼 Admin Dashboard and Management APIs
* 🗄️ MongoDB Database Integration

This backend serves as the core engine of the AI Study Material Generator, enabling secure user management, intelligent content generation, and efficient data processing.
